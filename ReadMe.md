# TSE Analytics - 2024

Projeto com finalidade de apresentar dados referentes à candidaturas dos partidos para as eleições municipais brasileiras de 2024.

Criamos um app no Streamlit que você pode [conferir aqui](https://tse-analytic.streamlit.app/).

# Fonte dos dados
Tribunal Superior Eleitoral (TSE): [Acesse os dados aqui](https://dadosabertos.tse.jus.br/dados/conjuntos-dados/candidatos-2024).
